##  How to Run a Project Locally
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the specific project folder:
   ```bash
   cd golang-projects/hello-world
   ```
3. Execute the code:
   ```bash
   go run main.go
   ```
